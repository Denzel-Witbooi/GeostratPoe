# 🗺️ GeoStrat

### 🌍 About the Project
**GeoStrat** is a location-based mobile application that helps users discover nearby landmarks, including monuments, parks, and tourist attractions. Designed with user-friendly navigation and a clean interface, GeoStrat leverages GPS and mapping technologies to provide real-time directions, landmark details, and location tracking.

Whether you're exploring a new city or discovering hidden gems in your hometown, GeoStrat guides your journey with ease.

## 🚀 Features

- 🔐 Secure user authentication (Sign Up, Login, Forgot Password)
- 📍 Current location and nearby landmarks
- 🗺️ Multiple map styles (Standard, Terrain, Satellite)
- 🚦 Real-time traffic overlay
- 📌 Custom markers and info windows
- 🧭 Directional routing (Walking, Biking, Driving, Transit)
- 💾 Save locations for quick access
- ⚙️ Profile management (Update username, email, password, profile image)
- 🌐 Unit system toggle (Metric ↔ Imperial)
- 🔎 Search for specific locations
- 🚪 Easy logout functionality

---

## 🛠️ Technologies Used

### 📱 Mobile Development
- Android SDK (Java)
- Google Maps SDK for Android
- Places API
- Directions API

### 🔐 Authentication & Database
- Firebase Authentication
- Firebase Realtime Database
- Firebase Cloud Storage

### ☁️ Cloud & DevOps
- Firebase Hosting & Monitoring
- GitHub for version control
